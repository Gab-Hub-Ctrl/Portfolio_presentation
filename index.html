Résumé IA
Note importante

INIT_SEQUENCE est une procédure architecturale conçue pour les IA de session à contexte persistant. Sur les IA publiques actuelles (GPT-4, Claude, Gemini), l'imposition d'un état externe est restreinte par les fournisseurs.

Sa valeur est démonstrative : structuration d'un système complexe, machine à états finis, gouvernance formalisée.
